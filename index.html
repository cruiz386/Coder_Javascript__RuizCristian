SIMULADOR - GESTOR DE TAREAS

Instrucciones:

Para el uso del gestor de tareas debe:

  1. Ingresar numero de usuario - usuario simulador 606060
  2. Ingresar clave de usuario - Clave12345
  3. Maximo de 3 intentos de usuario y/o clave incorrecta
  4. Ingresar accion que desea realizar:
  • A: Crear una tarea
  • B: Modificar una tarea
  • C: Eliminar una tarea
  • D: Mostrar tareas
  • ESC o Cancelar para SALIR
